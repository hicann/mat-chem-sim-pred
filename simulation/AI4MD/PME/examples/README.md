# PME/Ewald 长程静电算子 — 使用示例

## 单元测试

`test_pme.cpp` — **NPU vs CPU 精度对比 + 性能** 单元测试

### 测试项

| # | 测试 | 类型 | 对比基准 |
|---|------|------|----------|
| 1 | 初始化 | 功能 | — |
| 2 | 零电荷 NPU vs CPU | 精度 | CPU直接Coulomb求和 |
| 3 | 点电荷对 NPU vs CPU+解析 | 精度 | CPU + 解析解 E=q₁q₂/(4πε₀r) |
| 4 | 多电荷 (4原子) | 精度 | CPU O(N²)直接求和 |
| 5 | 性能基准 (64原子) | 性能 | 10次均值 |

### CPU参考实现

- 直接Coulomb求和: O(N²), 含最小镜像约定
- 解析解: E = 138.935485·q₁q₂/r (2原子无镜像)

### 精度要求

| 场景 | 容差 |
|------|------|
| 零电荷 | E=0, NPU vs CPU < 1e-4 |
| 点电荷对 | NPU vs 解析 < 1e-3 |
| 4原子多电荷 | NPU vs CPU < 1% |
