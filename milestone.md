# Milestone — 算子开发路线图

> 基于华为 CANN（Ascend C）框架，面向化工行业，在 **科学计算** 与 **预测优化** 两个方向上规划算子类型。
> 以下按 **算法出发** 组织，每个方向下设若干子类。

---

## 一、科学计算方向（Scientific Computing）

### 1.1 材料性质预测与材料结构生成

面向催化剂设计、电池材料、高分子材料等场景，基于第一性原理数据或经典力场数据，构建材料性质快速预测与结构逆向生成能力。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **材料晶体图神经网络** | CGCNN / MEGNet / GATGNN 等晶体图模型：原子为节点、键为边的消息传递网络，预测带隙、形成能、体积模量、热容等晶格性质 | 规划中 |
| **材料描述符批量化计算** | 组分描述符（元素比例、电负性差）、结构描述符（配位数、多面体扭曲度）、电子描述符（d 带中心、Bader 电荷）的高通量计算 | 规划中 |
| **晶体结构生成模型** | Diffusion 模型 / Flow 模型驱动的周期性结构生成：给定元素组成和空间群约束，生成稳定晶格参数与原子坐标 | 规划中 |
| **相图预测与相变识别** | 聚类 + 热力学模型（CALPHAD 替代模型）：从 MD 或 DFT 数据中识别相边界、计算相图 | 规划中 |
| **聚合物性质预测** | 聚合物图网络（Polymet / GNN for polymer）：重复单元序列 → 玻璃化温度、介电常数、气体渗透率 | 规划中 |
| **配方-结构-性能关联** | 多模态融合模型：将配方组成（表格）+ 工艺参数 + 微观结构图像联合嵌入，回归宏观性能 | 规划中 |

### 1.2 机器学习分子动力学（ML-MD）

以 ML 势函数替代经典力场，在保持 DFT 精度的同时实现大体系长时间 MD 模拟。此方向算子的核心是 **势能推理算子**（能量 + 力 + 应力）与 **在线学习算子**。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **DeepMD 风格推理算子** | 基于嵌入网络（embedding net）→ 拟合网络（fitting net）的原子势能推理：输入原子坐标和种类，输出总能量与原子上力。算子化实现 embedding 矩阵构建与网络前向传播 | 规划中 |
| **SchNet/PaiNN 推理算子** | 连续滤波器卷积（SchNet）或极性等变消息传递（PaiNN）的推理过程：含距离基函数展开、消息传递层、读出层，同时支持偶极/极化率预测 | 规划中 |
| **MACE/NequIP 推理算子** | 多体等变图神经网络推理：利用不可约张量积构造等变消息，适合高精度势能面拟合。算子化难点在于球形谐波和 Clebsch‑Gordan 系数的加速计算 | 规划中 |
| **ANI 风格推理算子** | 原子中心神经网络（AENet）推理：每个原子独立过 AEV（原子环境向量）→ MLP → 能量贡献，求和得总能量。算子化关键在于 AEV 的高效并行计算 | 规划中 |
| **描述符在线计算算子** | 在 MD 轨迹运行中实时计算 SOAP / ACSF / Behler‑Parrinello 对称函数，用于主动学习（active learning）中的不确定性探测 | 规划中 |
| **D3 色散校正算子** | Grimme D3 / D4 色散校正：基于原子坐标计算两体/三体色散项和阻尼函数，作为 ML 势的物理先验修正 | 规划中 |
| **势能面在线拟合算子** | 稀疏高斯过程（sGP）或贝叶斯线性回归：在 MD 运行中增量更新势能面模型，实现 "on‑the‑fly" 训练 | 规划中 |

### 1.3 AI for PDE

面向流体力学、传热传质、结构力学等化工过程物理场模拟场景，用 AI 方法加速或替代传统 PDE 求解器。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **PINN 推理算子** | 物理信息神经网络（PINN）推理：全连接网络 + 自动微分输出场变量（速度场、压力场、温度场）。算子化实现网络前向 + 梯度计算 | 规划中 |
| **FNO 推理算子** | 傅里叶神经算子（FNO）推理：在频域学习 PDE 解算子映射，含 FFT → 频域线性变换 → IFFT 的完整链路 | 规划中 |
| **DeepONet 推理算子** | 深度算子网络推理：Branch net（编码输入函数）→ Trunk net（编码查询位置）→ 内积输出解场 | 规划中 |
| **MeshGraphNet 推理算子** | 非结构网格上的图神经网络 PDE 求解：网格节点为图节点，边编码空间关系，消息传递预测物理场 | 规划中 |
| **可微分 PDE 层算子** | 将经典 PDE 离散格式（有限差分、有限体积）封装为可微分层，嵌入神经网络中进行端到端学习 | 规划中 |
| **CFD 降阶模型算子** | POD‑RNN / Autoencoder‑LSTM：对 CFD 高维仿真数据进行降维，构建时间序列预测模型，实现流场的快速推理 | 规划中 |

## 二、预测优化方向（Prediction & Optimization）

### 2.1 表格类数据预测预训练模型

面向化工实验/工艺数据中常见的表格型结构化数据（配方表、物性表、操作条件表），构建通用的预训练 + 微调框架。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **TabTransformer 推理算子** | 将表格的类别特征通过 Transformer 做 Embedding，数值特征直接拼接，构成混合编码器，适合下游分类/回归任务 | 规划中 |
| **FT‑Transformer 推理算子** | 纯 Transformer 架构的表格模型：所有特征（类别 + 数值）经过 Embedding 层后输入 Transformer encoder，输出预测结果 | 规划中 |
| **TabNet 推理算子** | 基于稀疏注意力（Sparse Attention）的表格神经网络：使用 Transformer 风格编码器 + 特征选择掩码，自带可解释性 | 规划中 |
| **Gradient Boosting 替代算子** | 用神经网络近似 XGBoost / LightGBM 的决策树集成：DeepGBM / NODE（Neural Oblivious Decision Ensemble）等可微树模型 | 规划中 |
| **表格数据预训练算子** | 大规模化工表格上的预训练：Mask 列预测（类似 BERT）或对比学习（SimCLR 风格），产出可迁移的表征向量 | 规划中 |
| **缺失值生成算子** | 基于扩散模型或 VAE 的表格数据缺失值填充：不局限于简单插值，而是生成分布一致的合理值 | 规划中 |

### 2.2 时序类数据预测预训练模型

面向化工过程数据（DCS 历史数据、传感器流、批次轨迹）的时序建模与预训练。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **TimesNet 推理算子** | 将 1D 时间序列通过 FFT 变换为 2D 张量（多个频率周期的叠加），再用 2D 卷积捕获周期内与周期间的依赖 | 规划中 |
| **PatchTST 推理算子** | 将时间序列切分为子序列片段（Patch），输入 Transformer 进行自回归或掩码预训练。适合长序列预测 | 规划中 |
| **TimesFM 推理算子** | Google 提出的时序基础模型：Decoder‑only 架构，在大规模时序数据上预训练，支持零样本迁移到新时序场景 | 规划中 |
| **Mamba/SSM 推理算子** | 状态空间模型（Mamba 架构）处理时序：线性复杂度、长程依赖捕获。适合超长化工过程序列 | 规划中 |
| **时序预训练算子** | 大规模化工 DCS 时序上的预训练：对比学习（TS2Vec / TNC）或掩码重建（PatchTST 方式），产出时序 Embedding | 规划中 |
| **异常检测算子** | 基于 Autoencoder / VAE / Transformer 的时序异常检测：重建误差或预测误差作为异常得分，支持多变量工艺监控 | 规划中 |

### 2.3 小数据预测优化模型

化工领域常面临标记数据稀少（昂贵实验、长周期测试）的问题，此子类聚焦小样本、主动学习、贝叶斯优化等方法。

| 算子/模型类型 | 算法描述 | 开发阶段 |
|--------------|----------|----------|
| **高斯过程回归算子** | 经典 GP 回归：支持 RBF / Matern / 周期核等核函数，输出预测均值与不确定性。算子化实现 Cholesky 分解/共轭梯度求解 | 规划中 |
| **贝叶斯神经网络算子** | 基于 Dropout 变分推断或 Laplace 近似的 BNN：在保证预测精度的同时输出模型不确定性 | 规划中 |
| **主动学习采样算子** | 不确定性采样（熵、边际、BALD）/ 多样性采样（CoreSet / TypClust）：从候选池中挑选最有标注价值的数据 | 规划中 |
| **贝叶斯优化算子** | 以 GP 或 RF 为代理模型的贝叶斯优化：采集函数（EI / PI / UCB）计算，支持带约束的多目标优化 | 规划中 |
| **小样本迁移学习算子** | 预训练 → 微调 pipeline：从源域（公开数据集/模拟数据）预训练模型，通过少量目标域数据微调适配 | 规划中 |
| **度量学习/原型网络算子** | Prototypical Network / Siamese Network：在嵌入空间中通过距离度量实现小样本分类（如故障类型识别） | 规划中 |
| **数据增强算子** | 基于 VAE / GAN / Diffusion 的化工数据增强：从少量真实样本出发，生成分布一致的合成样本 | 规划中 |

## 三、其他数据与模型类型（参考，非当前 Milestone）

以下类型在化工 AI 中同样具有重要价值，但受限于硬件适配、场景清晰度或资源投入，暂不作为本里程碑的优先开发目标，留待后续评估。

| 类别 | 典型模型/算法 | 暂缓原因 | 后续评估时机 |
|------|--------------|----------|-------------|
| **图像与视觉模型** | CNN / ViT / ResNet / U‑Net（催化剂显微图像、火焰图像、流型图像分类与分割） | 偏视觉感知，化工专用属性不强，通用 CV 框架已有成熟方案 | 当出现强化工视觉需求 + CANN 原生图像算子打通时 |
| **图数据通用模型** | GCN / GAT / GraphSAGE（分子图通用建模） | 已有的工业图框架（如 DGL、PyG）对 CANN 适配度待验证 | CANN 图框架成熟后评估迁移成本 |
| **大语言模型（LLM）** | LLaMA / ChatGLM / Bloom（化工文献挖掘、安全问答、报告生成） | 对显存和算子规模要求高，CANN 当前版本以计算为主而非文本 | 团队明确 NLP 化工场景需求后单独立项 |
| **多模态融合通用框架** | CLIP / ViLT 等图文多模态（实验装置图 + 文本描述 + 数值表联合建模） | 场景尚未收敛，且涉及视觉/文本/数值异质数据 | 具体化工多模态任务出现后再规划 |
| **强化学习通用框架** | PPO / SAC / DQN（过程控制、调度优化） | 通用 RL 框架可对接，无需自定义算子；若需加速可考虑后续 | 考虑将 RL 环境仿真部分纳入 AI for PDE 方向 |
| **语音与音频模型** | Whisper / Wav2Vec（化工设备声音监控与故障诊断） | 非主流化工 AI 场景，暂不纳入规划 | 设备声音监控需求明确后评估 |
| **物理仿真传统求解器** | FVM / FEM / LBM 传统求解器（非 AI 的纯数值 PDE 求解器） | 不属 AI 算子范畴，与 AI for PDE 方向互补而非从属 | 如需混合 AI+ 传统求解，可转为 AI for PDE 的辅助模块 |

## 四、优先级策略

| 优先级 | 含义 | 建议资源占比 |
|--------|------|-------------|
| **科学计算（材料预测 + ML‑MD + AI for PDE）** | 已有 MD 链路基础，补齐 ML 势和 PDE 求解能力，形成科学计算闭环 | 约 50% |
| **预测优化（表格 + 时序 + 小数据）** | 覆盖化工行业最普遍的数据形态，产出可直接落地的预测与优化能力 | 约 40% |
| **其他类别** | 保留扩展性，待条件成熟后评估进入里程碑 | 约 10%（探索性投入） |

> **核心原则**：以科学计算为基座（材料/分子模拟数据产出），以预测优化为目标（将数据转化为可决策的模型），构建 "**模拟 → 数据 → 预测 → 设计 → 优化**" 的全链路化工 AI 算子体系。
