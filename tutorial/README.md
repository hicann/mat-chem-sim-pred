# Tutorial / 教程

本目录包含基于 Jupyter Notebook 的教学教程，帮助开发者快速了解本仓库的算子体系并上手使用。

## 适用人群

- 希望了解 **CANN Ascend C** 算子开发的初学者
- 对 **化工 + AI** 交叉领域感兴趣的研究人员和工程师
- 希望将 **分子模拟 / PDE 求解 / 表格预测 / 时序建模 / 小数据优化** 迁移到昇腾 NPU 的开发者

## 教程目录

| 编号 | 名称 | 内容 | 所需前置 |
|------|------|------|----------|
| 00 | **环境准备** | 验证 Python/PyTorch/CANN 环境，克隆仓库，跑通第一个测试 | 无 |
| 01 | **Ascend C 基础** | 算子结构、模板解读、构建流程 | 教程 00 |
| 02 | **材料性质预测** | 晶体图表示、DAO 算子库使用、基础性质预测 | 教程 01 |
| 03 | **机器学习分子动力学** | LJ 力场、MD 模拟流程、Ascend C 算子调用 | 教程 01 |
| 04 | **AI for PDE** | PINN/FNO/DeepONet 原理、算子演示 | 教程 01 |
| 05 | **表格类数据预测** | TabNet 模型使用、特征选择可视化 | 教程 00 |
| 06 | **时序类数据预测** | TimesNet 周期发现、预测实战 | 教程 00 |
| 07 | **小数据预测与优化** | 高斯过程回归、贝叶斯优化实战 | 教程 00 |

## 使用方式

### 方式一：本地 Jupyter

```bash
# 安装依赖
pip install jupyter torch numpy matplotlib pandas

# 启动
cd tutorial
jupyter notebook
```

### 方式二：华为开发者空间 AI NoteBook

1. 登录 [华为开发者空间](https://developer.huaweicloud.com/space)
2. 创建 Notebook 实例，选择 `Ascend-Powered-Engine` 镜像
3. 克隆本仓库后进入 `tutorial/` 目录
4. 打开对应 `.ipynb` 文件运行

## 代码风格

每个 notebook 遵循统一结构：

1. **学习目标** ⭐ — 明确本节学什么
2. **理论铺垫** 📖 — 必要的背景知识
3. **代码实践** 💻 — 可直接运行的代码
4. **结果可视化** 📊 — 图表展示结果
5. **思考练习** ✏️ — 巩固理解的思考题
